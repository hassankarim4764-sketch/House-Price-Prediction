# House Price Prediction Using Data Mining & Machine Learning

## Overview

This project focuses on predicting house prices using Data Mining and Machine Learning techniques. The system analyzes housing data, performs preprocessing, and applies predictive models to estimate property prices accurately.

## Objectives

* Analyze housing market data
* Perform data preprocessing and cleaning
* Visualize important patterns and relationships
* Train machine learning models for price prediction
* Evaluate prediction accuracy

## Technologies Used

* Python
* Pandas
* NumPy
* Scikit-learn
* Matplotlib
* Seaborn
* Jupyter Notebook

## Machine Learning Techniques

* Regression Analysis
* Data Preprocessing
* Feature Engineering
* Data Visualization

## Features

* Data Cleaning

* Graphical Analysis
* Price Prediction
* Model Evaluation

## Project Files

The project successfully predicts house prices using machine learning algorithms and data mining techniques with effective data analysis and visualization.
